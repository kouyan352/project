## 商品の追加方法

### ステップ1: 商品管理画面へ移動
```
Shopify管理画面
→ 左メニュー「商品管理」
→ 「商品を追加する」をクリック
```

### ステップ2: 基本情報を入力

#### 必須項目
- **タイトル**: 商品名（例: 「モンステラ 中型」）
- **説明**: 商品の詳細説明
- **価格**: 販売価格

#### 推奨項目
- **比較価格**: 元の価格（セール時に便利）
- **在庫管理**: 在庫数を追跡
- **SKU**: 商品コード（任意）

### ステップ3: 画像をアップロード
```
「メディア」セクション
→ 「ファイルを追加する」
→ 画像を選択（最大10枚）
→ ドラッグで順序変更可能
```

**推奨画像サイズ**: 800x800px 以上（正方形）

### ステップ4: 保存
```
右上の「保存する」をクリック
```

## 商品情報の編集

### タイトルと説明の変更

#### 方法1: 管理画面から
```
商品管理
→ 編集したい商品をクリック
→ タイトルや説明を変更
→ 保存
```

#### 方法2: 一括編集
```
商品管理
→ 複数商品にチェック
→ 「アクション」→「商品を編集」
→ 一括で変更
```

### 価格の変更

```
商品編集画面
→ 「価格設定」セクション
→ 価格を変更
→ 保存
```

**セール価格の設定**:
- 「比較価格」に元の価格を入力
- 「価格」に割引後の価格を入力
- 自動的に打ち消し線が表示されます

## 商品画像の管理

### 画像の追加
```
商品編集画面
→ 「メディア」セクション
→ 「ファイルを追加する」
→ 画像を選択
```

### 画像の順序変更
```
画像をドラッグ＆ドロップ
→ 一番左が商品一覧で表示される画像
```

### 画像の削除
```
画像にマウスをホバー
→ ゴミ箱アイコンをクリック
```

### 画像の代替テキスト（SEO）
```
画像をクリック
→ 「代替テキストを追加」
→ 画像の説明を入力（例: 「モンステラの葉の詳細」）
```

## 商品バリエーションの設定

商品に複数のオプション（サイズ、色など）を追加する方法

### ステップ1: オプションを追加
```
商品編集画面
→ 「バリエーション」セクション
→ 「オプションを追加する」
```

### ステップ2: オプションの設定

#### 例1: サイズバリエーション
```
オプション名: サイズ
値: 小型, 中型, 大型
```

#### 例2: 色バリエーション
```
オプション名: 鉢の色
値: 白, グレー, テラコッタ
```

### ステップ3: バリエーションごとの設定
```
各バリエーションで個別に設定可能:
- 価格
- 在庫数
- SKU
- 画像
```

### 具体例
```
商品: モンステラ

バリエーション1:
- サイズ: 小型
- 価格: 3,000円
- 在庫: 10個

バリエーション2:
- サイズ: 中型
- 価格: 5,000円
- 在庫: 5個

バリエーション3:
- サイズ: 大型
- 価格: 8,000円
- 在庫: 3個
```

## コレクション（カテゴリー）の作成

商品をグループ化して表示する方法

### ステップ1: コレクション作成
```
Shopify管理画面
→ 「商品管理」→「コレクション」
→ 「コレクションを作成する」
```

### ステップ2: コレクション情報
```
タイトル: 「観葉植物」など
説明: コレクションの説明文
```

### ステップ3: コレクションタイプ選択

#### 手動コレクション
```
「手動」を選択
→ 商品を手動で追加
→ 順序を自由に変更可能
```

#### 自動コレクション
```
「自動」を選択
→ 条件を設定（例: タグに「植物」を含む）
→ 条件に合う商品が自動追加
```

### コレクション例
```
コレクション1: 「初心者向け植物」
- 条件: タグに「初心者」を含む
- 自動で該当商品を表示

コレクション2: 「人気商品」
- 手動で選択
- おすすめ順に並べる

コレクション3: 「セール商品」
- 条件: 比較価格が設定されている
- 自動でセール品を表示
```

## 商品ページのカスタマイズ

Green Care Themeでの商品ページカスタマイズ方法

### 方法1: テーマエディタで編集

```
オンラインストア
→ テーマ → カスタマイズ
→ ページを選択: 「商品」
→ セクションを編集
```

### 編集可能な要素
- 商品画像のレイアウト
- 商品情報の表示順序
- おすすめ商品の表示/非表示

### 方法2: コードで編集（上級者向け）

#### 商品情報セクションの編集
```
ファイル: sections/product-info.liquid
場所: Shopify管理画面 → テーマ → コードを編集
```

**編集できる内容**:
- レイアウト（画像の配置など）
- ボタンのテキスト
- 価格の表示形式
- 商品説明のスタイル

#### おすすめ商品セクションの編集
```
ファイル: sections/product-recommendations.liquid
```

**編集できる内容**:
- 表示商品数（デフォルト: 4個）
- グリッドレイアウト
- スタイル

## よくある編集タスク

### タスク1: カートボタンのテキスト変更

#### 現在
```
「カートに追加」
```

#### 変更方法
```
1. テーマ → コードを編集
2. sections/product-info.liquid を開く
3. 検索: "カートに追加"
4. 好きなテキストに変更（例: "購入する"）
5. 保存
```

### タスク2: 商品画像のサイズ変更

#### ファイル
```
sections/product-info.liquid
```

#### 変更箇所
```html
<!-- 現在: 800x800 -->
<img src="{{ product.featured_image | img_url: '800x800' }}">

<!-- 変更例: 1000x1000 -->
<img src="{{ product.featured_image | img_url: '1000x1000' }}">
```

### タスク3: 商品一覧の表示数変更

#### ファイル
```
sections/collection-products.liquid
```

#### 変更箇所
```liquid
<!-- 現在: 1ページ12個 -->
{% paginate collection.products by 12 %}

<!-- 変更例: 1ページ24個 -->
{% paginate collection.products by 24 %}
```

### タスク4: おすすめ商品の表示数変更

#### ファイル
```
sections/product-recommendations.liquid
```

#### 変更箇所
```liquid
<!-- 現在: 4個 -->
{% for product in recommendations.products limit: 4 %}

<!-- 変更例: 6個 -->
{% for product in recommendations.products limit: 6 %}
```

#### CSSも変更（グリッド）
```css
/* 現在: 4列 */
.recommendations-grid {
  grid-template-columns: repeat(4, 1fr);
}

/* 変更例: 6列 */
.recommendations-grid {
  grid-template-columns: repeat(6, 1fr);
}
```

## 在庫管理

### 在庫追跡の有効化
```
商品編集画面
→ 「在庫」セクション
→ 「Shopifyで在庫を追跡する」にチェック
→ 在庫数を入力
```

### 在庫切れの表示
```
在庫が0になると自動的に:
- カートボタンが「売り切れ」に変更
- ボタンが無効化
```

### 在庫アラート
```
設定 → 通知
→ 在庫が少なくなったら通知
→ しきい値を設定（例: 5個以下）
```

## SEO最適化

### 商品ページのSEO設定

#### 基本設定
```
商品編集画面
→ 一番下の「検索エンジンのリスト」
→ 「ウェブサイトのSEOを編集する」
```

#### 編集項目
1. **ページタイトル**
   - 例: 「モンステラ（中型）| Green Care」
   - 60文字以内推奨

2. **メタディスクリプション**
   - 例: 「初心者にもおすすめのモンステラ。育てやすく、お部屋に緑を添えます。」
   - 160文字以内推奨

3. **URL**
   - 例: `/products/monstera-medium`
   - 英数字とハイフン推奨

## トラブルシューティング

### Q: 商品が表示されない
**A**: 以下を確認
1. 商品が「公開」状態か確認
2. 在庫が0でないか確認
3. オンラインストアで利用可能になっているか確認

### Q: 画像が粗い
**A**: 
- 最低800x800px以上の画像を使用
- JPGまたはPNG形式
- ファイルサイズは2MB以下推奨

### Q: バリエーションが表示されない
**A**:
1. バリエーションを保存したか確認
2. 各バリエーションに在庫があるか確認
3. テーマが最新版か確認

### Q: おすすめ商品が表示されない
**A**:
- Shopifyのおすすめアルゴリズムは購入履歴に基づく
- 新しいストアでは表示されない場合がある
- 関連商品のタグを追加すると改善

## 商品編集のベストプラクティス

### ✅ やるべきこと
1. **高品質な画像を使用**
   - 明るく、ピントが合っている
   - 複数角度から撮影
   - 白背景推奨

2. **詳細な説明を書く**
   - 商品の特徴
   - サイズ・重量
   - お手入れ方法
   - 配送情報

3. **適切なタグをつける**
   - 検索性向上
   - コレクション管理が楽

4. **SEO設定をする**
   - ページタイトル
   - メタディスクリプション
   - ALTテキスト

5. **在庫管理を有効化**
   - 売り切れを防ぐ
   - 在庫数の把握

### ❌ 避けるべきこと
1. 低品質な画像
2. 説明が不十分
3. 価格が未設定
4. タグなし
5. SEO未設定
