### 手順

```bash

```

- オンラインストア → ページ → ページを追加

#### 3. HTMLモードに切り替え
- 右上の「<>」アイコンをクリック

#### 4. コピー&ペースト
- `shopify-page-complete.html` の内容を全てコピー
- Shopifyのエディタに貼り付け

#### 5. 保存して公開
- 「保存」ボタンをクリック
- 完成！🎉

### LINE URLの変更

ファイル内の全ての `https://lin.ee/8QakieO` を検索して、
あなたのLINE公式アカウントURLに置き換えてください。

```html
<!-- 変更前 -->
<a href="https://lin.ee/8QakieO" ...>

<!-- 変更後 -->
<a href="https://lin.ee/YOUR_LINE_ID" ...>
```

### 色の変更

CSSセクション（ファイル上部）で色を変更:

```css
.green-care-page {
    --primary-color: #2D5F3F;    /* メインの緑色 */
    --secondary-color: #4A7C59;  /* サブカラー */
    --accent-color: #7BA885;     /* アクセント */
}
```

HTMLセクションで直接編集:

## 🔍 トラブルシューティング

### 問題: スタイルが正しく適用されない

**解決方法:**
1. ブラウザのキャッシュをクリア（Ctrl+Shift+R）
2. Shopifyテーマエディタで「プレビュー」を確認
3. クラス名の競合がないか確認（`gc-` プレフィックス使用）

### 問題: フォントが表示されない

**解決方法:**
1. インターネット接続を確認
2. Google Fontsがブロックされていないか確認
3. 必要に応じて代替フォントを設定

### 問題: レスポンシブが機能しない

**解決方法:**
1. Shopifyテーマの設定を確認
2. メタタグが正しく設定されているか確認
3. ブラウザの開発者ツールで確認

### よくある質問

**Q: 他のShopifyテーマでも使えますか？**  
A: はい、クラス名にプレフィックスを使用しているため、ほとんどのテーマと互換性があります。

**Q: 無料テーマでも動作しますか？**  
A: はい、Shopifyの標準機能のみを使用しています。

**Q: 日本語以外にも対応できますか？**  
A: はい、HTML内のテキストを翻訳すれば多言語対応可能です。

**Q: パフォーマンスへの影響は？**  
A: 最適化されたCSSのみを使用しており、パフォーマンスへの影響は最小限です。
