## 方法1: Shopifyページとして追加（推奨）⭐

### 所要時間: 約5分

この方法が最も簡単で、コーディング知識も不要です。

### ステップ1: Shopify管理画面にログイン

1. Shopifyストアの管理画面にアクセス
2. 左サイドバーから **「オンラインストア」** をクリック
3. **「ページ」** を選択

### ステップ2: 新しいページを作成

1. 右上の **「ページを追加」** ボタンをクリック
2. **タイトル** を入力（例: `Green Care - 植物サポートサービス`）
3. コンテンツエディタの右上にある **「<>」アイコン（HTMLを表示）** をクリック

![Shopify HTMLエディタ](https://i.imgur.com/example1.png)

### ステップ3: HTMLコードを貼り付け

1. `shopify-page-complete.html` ファイルを開く
2. **全ての内容をコピー** (Ctrl+A → Ctrl+C または Cmd+A → Cmd+C)
3. Shopifyのテキストエディタに **貼り付け** (Ctrl+V または Cmd+V)
4. 右上の **「保存」** ボタンをクリック

### ステップ4: ページを公開

1. ページの設定で「表示設定」を **「表示」** に変更
2. 必要に応じてSEO設定（メタディスクリプションなど）を入力
3. **「保存」** をクリック

### ステップ5: 確認

1. 「ページを表示」をクリックして確認
2. 問題なければ公開完了！

## 方法2: カスタムページテンプレート

### 所要時間: 約15分

より高度な統合を行いたい場合の方法です。

### ステップ1: テーマエディタにアクセス

1. Shopify管理画面 → **「オンラインストア」** → **「テーマ」**
2. 現在のテーマの **「アクション」** → **「コードを編集」**

### ステップ2: 新しいテンプレートを作成

1. 左サイドバーの **「Templates」** セクション
2. **「新しいテンプレートを追加」** をクリック
3. タイプ: **`page`** を選択
4. 名前: **`green-care`** と入力
5. **「作成」** をクリック

### ステップ3: コードを貼り付け

以下のコードを `templates/page.green-care.liquid` に貼り付け:

```liquid
{% layout 'theme' %}

<!-- shopify-page-complete.htmlの内容をここに貼り付け -->
```

### ステップ4: ページにテンプレートを適用

1. **「オンラインストア」** → **「ページ」**
2. 新しいページを作成
3. 右サイドバーの **「テンプレート」** セクション
4. `page.green-care` を選択
5. **「保存」**

## 🔧 カスタマイズ方法

### LINE URLを変更する

ファイル内の全ての `https://lin.ee/8QakieO` を、あなたの実際のLINE公式アカウントURLに置き換えてください。

**検索と置換の方法:**
1. テキストエディタで Ctrl+H (Windows) または Cmd+F (Mac)
2. 検索: `https://lin.ee/8QakieO`
3. 置換: `あなたのLINE URL`
4. 全て置換

### 色を変更する

CSSの`:root`セクション（ファイルの上部）で色を変更できます:

```css
:root {
    --primary-color: #2D5F3F;    /* メインカラー */
    --secondary-color: #4A7C59;  /* セカンダリーカラー */
    --accent-color: #7BA885;     /* アクセントカラー */
}
```

### テキストを変更する

HTMLセクションで直接テキストを編集できます。例:

```html
<h1 class="gc-hero-title">
    <span class="gc-title-small">水やりもSOSも、</span>
    <span class="gc-title-large">すべてスマホで。</span>
</h1>
```

### 料金プランの変更

料金プランセクションで価格や機能を変更:

```html
<span class="gc-price-amount">¥980</span>
```

## ❓ よくある質問

### Q1: ページが正しく表示されません

**A:** 以下を確認してください:
- HTMLコード全体がコピーされているか
- Shopifyのテーマと競合していないか
- ブラウザのキャッシュをクリアしてみてください (Ctrl+Shift+R)

### Q2: レスポンシブデザインは対応していますか？

**A:** はい、完全対応しています。モバイル、タブレット、デスクトップで最適化されています。

### Q3: 既存のテーマと統合できますか？

**A:** はい、クラス名にプレフィックス（`gc-`）を使用しているため、既存のテーマと競合しにくい設計です。

### Q4: フォントが表示されません

**A:** Google Fontsを使用しています。インターネット接続を確認してください。
または、Shopifyテーマのフォントを使用したい場合は、`<link>` タグのフォント読み込み部分を削除してください。

### Q5: JavaScriptは必要ですか？

**A:** このバージョンは主にCSSアニメーションを使用しているため、基本的な表示にJavaScriptは不要です。
ただし、より高度なインタラクション機能が必要な場合は、元の `script.js` を追加できます。

### Q6: SEO対策はどうすればいいですか？

**A:** Shopifyのページ設定で以下を設定してください:
- ページタイトル: "Green Care - 観葉植物LINEサポートサービス"
- メタディスクリプション: "観葉植物の育て方をLINEでサポート。水やりリマインダー、専門家相談、24時間以内回答。"
- URLハンドル: "green-care" または "plant-support"

### Q7: ページの読み込みが遅いです

**A:** 以下を試してください:
- Shopifyのテーマ最適化機能を使用
- 不要な画像を削除または圧縮
- Shopify Apps の数を減らす

### Q8: ホームページとして設定できますか？

**A:** はい、以下の手順で設定できます:
1. **「オンラインストア」** → **「テーマ」** → **「カスタマイズ」**
2. **「ホームページ」** セクション
3. **「セクションを追加」** → **「カスタムコンテンツ」**
4. HTMLコードを貼り付け
