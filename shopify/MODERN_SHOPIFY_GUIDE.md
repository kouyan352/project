## 🚀 5分で完了！導入手順

### ステップ1️⃣: ファイルを開く

`/home/user/webapp/shopify/shopify-modern-complete.html`

このファイルを開いて、**全内容をコピー**してください。

### ステップ2️⃣: Shopify管理画面にアクセス

1. Shopifyストアの管理画面にログイン
2. 左メニューから **「オンラインストア」** をクリック
3. **「ページ」** を選択
4. 右上の **「ページを追加」** ボタンをクリック

### ステップ3️⃣: ページ情報を入力

- **タイトル**: `Green Care - 植物サポートサービス`
- **説明**: 必要に応じて入力

### ステップ4️⃣: HTMLモードに切り替え

1. コンテンツエディタの右上にある **「<>」アイコン** をクリック
2. HTMLエディタモードに切り替わります

### ステップ5️⃣: コードを貼り付け

1. `shopify-modern-complete.html` の**全内容をコピー**
2. Shopifyのテキストエディタに**貼り付け**（Ctrl+V / Cmd+V）

### ステップ6️⃣: 保存して公開

1. 右上の **「保存」** ボタンをクリック
2. ページの公開設定で **「表示」** を選択
3. **「保存」** して完了！

## 🔧 カスタマイズ方法

### LINE URLの変更

ファイル内の全ての `https://lin.ee/8QakieO` を、
あなたのLINE公式アカウントURLに置き換えます。

**方法1: テキストエディタで一括置換**
```
検索: https://lin.ee/8QakieO
置換: https://lin.ee/YOUR_LINE_ID
```

**方法2: Shopify内で直接編集**
1. ページを保存後、再度編集
2. HTMLビューで検索（Ctrl+F / Cmd+F）
3. 一つずつ置換

### 色の変更

CSSセクション（ファイル上部）で色を変更：

```css
.green-care-modern {
    --primary-color: #2D5F3F;    /* メインの緑色 */
    --secondary-color: #4A7C59;  /* サブカラー */
    --bright-green: #2ECC71;     /* 明るい緑 */
}
```

### テキストの変更

HTMLセクションで直接編集：

```html
<!-- 変更前 -->
<span class="gc-title-large">すべてスマホで。</span>

<!-- 変更後 -->
<span class="gc-title-large">あなたのテキスト</span>
```

## 🔍 トラブルシューティング

### 問題: デザインが崩れる

**原因:** Shopifyテーマとのスタイル競合

**解決策:**
1. クラス名にプレフィックス（`gc-`）使用済み
2. Shopifyテーマのカスタムスタイルを確認
3. ブラウザの開発者ツールで調査

### 問題: アニメーションが重い

**原因:** デバイスの性能またはブラウザ

**解決策:**
1. 古いデバイスでは一部アニメーション無効化
2. CSSの`animation`を`none`に変更
3. JavaScriptのアニメーションをコメントアウト

### 問題: ガラス効果が表示されない

**原因:** 古いブラウザでは非対応

**解決策:**
- 自動的にフォールバック（通常の背景）に切り替わります
- IE11では一部効果が表示されませんが、機能は正常です
