# Green Care - 観葉植物ライフサポート

スマホで始める、新しい植物ライフ。LINEで専門家に相談できる観葉植物サポートサービスのランディングページです。

## 🌱 プロジェクト概要

Green Careは、観葉植物の育成に悩む方々をLINEでサポートするサービスのウェブサイトです。専門家への相談、水やりリマインダー、成長記録管理など、植物ライフを豊かにする機能を提供します。

## ✨ 特徴

- **モダンなデザイン**: グリーンを基調とした落ち着いたカラーパレット
- **レスポンシブ対応**: モバイル、タブレット、デスクトップに完全対応
- **スムーズなアニメーション**: スクロールに応じた要素の表示アニメーション
- **インタラクティブUI**: ホバーエフェクト、リップルエフェクトなど
- **高速パフォーマンス**: 最適化されたCSS、JavaScript

## 🚀 使い方

### ローカル環境での実行

1. このディレクトリに移動:
```bash
cd /home/user/webapp
```

2. ローカルサーバーを起動:
```bash
# Python 3の場合
python3 -m http.server 8000

# Python 2の場合
python -m SimpleHTTPServer 8000

# Node.jsのhttp-serverを使う場合
npx http-server -p 8000
```

3. ブラウザで開く:
```
http://localhost:8000
```

## 🔧 カスタマイズ

### 色の変更
`style.css`の`:root`セクションでカラー変数を変更できます:

```css
:root {
    --primary-color: #2D5F3F;
    --secondary-color: #4A7C59;
    /* その他の色 */
}
```
