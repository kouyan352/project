# 🎨 Shopifyテーマとしてインストールする方法

## 📦 green-care-shopify-theme.zip の使い方

このZIPファイルは **完全なShopifyテーマ** として作成されています。
Shopifyの管理画面から直接アップロードしてインストールできます。

---

## 🚀 インストール手順（推奨）

### ステップ1: Shopify管理画面にログイン
1. あなたのShopifyストアにログイン
2. 左メニューから **「オンラインストア」** をクリック

### ステップ2: テーマをアップロード
1. **「テーマ」** セクションに移動
2. ページを下にスクロールして **「テーマライブラリ」** を見つける
3. **「テーマをアップロード」** ボタンをクリック
4. `green-care-shopify-theme.zip` を選択してアップロード
5. アップロードが完了するまで待つ（通常30秒〜1分）

### ステップ3: テーマをプレビュー
1. アップロードされた **「Green Care Theme」** を見つける
2. **「アクション」** → **「プレビュー」** をクリック
3. 新しいタブでプレビューが開きます
4. デザインを確認

### ステップ4: テーマを公開（オプション）
- **注意**: 既存のテーマを置き換えます
- **「アクション」** → **「公開する」** をクリック
- 確認ダイアログで **「公開する」** をクリック

---

## 📄 Green Careページの作成

テーマをインストールした後、専用ページを作成します：

### ステップ1: 新しいページを作成
1. Shopify管理画面で **「オンラインストア」** → **「ページ」**
2. **「ページを追加」** をクリック

### ステップ2: ページ設定
- **タイトル**: `Green Care - 植物サポート` など
- **テンプレート**: 右側のメニューから **「page.green-care」** を選択
- **テンプレートが見つからない場合**: デフォルトの「page」を使用してOK

### ステップ3: 保存して確認
- **「保存」** をクリック
- **「ページを表示」** でプレビュー確認

---

## ⚙️ カスタマイズ方法

### LINE URLの変更

#### 方法1: テーマエディタで変更（簡単）
1. **「オンラインストア」** → **「テーマ」**
2. Green Care Theme の **「カスタマイズ」** をクリック
3. 左サイドバーで **「Header」** セクションを選択
4. **「LINE URL」** フィールドに実際のURLを入力
5. **「保存」** をクリック

#### 方法2: コードで一括変更
1. **「オンラインストア」** → **「テーマ」**
2. Green Care Theme の **「アクション」** → **「コードを編集」**
3. 検索機能（Ctrl+F / Cmd+F）で `https://lin.ee/8QakieO` を検索
4. すべて実際のLINE URLに置換
5. **「保存」** をクリック

### カラーの変更
1. **「オンラインストア」** → **「テーマ」**
2. **「カスタマイズ」** をクリック
3. **「テーマ設定」** → **「Colors」**
4. Primary Color、Secondary Color、Accent Colorを変更
5. **「保存」** をクリック

### テキストの変更
1. **「オンラインストア」** → **「テーマ」** → **「カスタマイズ」**
2. ページ上の編集したいセクションをクリック
3. 左サイドバーでテキストを編集
4. **「保存」** をクリック

## 🎯 2つの使い方

### 方法A: メインテーマとして使用（推奨）
- テーマ全体をアップロードして公開
- サイト全体がGreen Careスタイルになる
- カスタマイズが簡単

### 方法B: 特定のページのみに使用
- 既存テーマはそのまま
- `page.green-care.liquid` テンプレートを作成した特定のページのみ適用
- 他のページは既存デザインを維持

## ❓ トラブルシューティング

### Q: テーマがアップロードできない
**A**: ZIPファイルが破損していない確認
- ファイルサイズが27KB程度あることを確認
- 再ダウンロードして再度試す

### Q: 「page.green-care」テンプレートが見つからない
**A**: テーマを再アップロード
- テーマを削除して再度アップロード
- または、デフォルトの「page」テンプレートを使用

### Q: アニメーションが動かない
**A**: JavaScriptが読み込まれているか確認
1. ブラウザの開発者ツールを開く（F12）
2. Consoleタブを確認
3. エラーがあれば報告してください

### Q: デザインが崩れる
**A**: ブラウザキャッシュをクリア
1. Ctrl+Shift+R（Windows）/ Cmd+Shift+R（Mac）で強制リロード
2. ブラウザキャッシュをクリア
3. それでも直らない場合は、別のブラウザで確認

### Q: 既存テーマを残したい
**A**: テーマを複製してからカスタマイズ
1. 現在のテーマの **「アクション」** → **「複製する」**
2. バックアップを作成
3. その後Green Care Themeをアップロード
