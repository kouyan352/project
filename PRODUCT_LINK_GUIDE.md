## 🎯 設定手順

### ステップ1: Shopifyで商品を作成

1. **Shopify管理画面** にログイン
   ```
   https://あなたのストア名.myshopify.com/admin
   ```

2. **左メニュー → 商品**

3. **商品を追加** をクリック

4. **3つのプラン商品を作成**:

#### 📦 フリープラン
```
商品名: Green Care フリープラン
価格: ¥0
商品ハンドル: free-plan （自動生成される）
説明: 
- 水やりリマインダー（1植物まで）
- チャットボット相談（無制限）
- 基本的な育て方ガイド

商品タイプ: サブスクリプション（アプリ使用）
```

#### 📦 スタンダードプラン
```
商品名: Green Care スタンダードプラン
価格: ¥980
商品ハンドル: standard-plan
説明:
- 水やりリマインダー（5植物まで）
- チャットボット相談（無制限）
- 専門家相談（月3回まで）
- 成長記録・写真保存
- 24時間以内の回答保証

商品タイプ: サブスクリプション
```

#### 📦 プレミアムプラン
```
商品名: Green Care プレミアムプラン
価格: ¥1,980
商品ハンドル: premium-plan

商品タイプ: サブスクリプション
```

### ステップ2: 商品ハンドルを確認

商品を保存すると、自動的に**商品ハンドル**が生成されます。

#### 確認方法:

1. 作成した商品を開く
2. URLを確認:
   ```
   https://admin.shopify.com/store/あなたのストア/products/【商品ハンドル】
   ```
3. 商品ハンドルをメモ

**例**:
- フリープラン → `free-plan`
- スタンダード → `standard-plan`
- プレミアム → `premium-plan`

## 📌 商品ハンドルの命名規則

Shopifyでは、商品ハンドルは以下のルールで自動生成されます：

- 商品名を小文字に変換
- スペースをハイフン（-）に変換
- 特殊文字を削除

**例**:
- `Green Care フリープラン` → `green-care-free-plan` または `free-plan`
- `スタンダードプラン` → `standard-plan`

手動で設定する場合：
1. 商品編集画面で **検索エンジンリスト** セクションを開く
2. **URLとハンドル** を編集

## 🚀 サブスクリプション機能の追加（オプション）

月額サブスクリプションを実装するには、Shopifyアプリが必要です：

### 推奨アプリ:

1. **Recharge Subscriptions**
   - 月額・定期購入に対応
   - 顧客管理機能

2. **Bold Subscriptions**
   - 柔軟なサブスクリプション設定
   - キャンセル・変更機能

3. **Seal Subscriptions**
   - シンプルなサブスクリプション機能
   - 日本語対応

### 設定方法:

1. Shopifyアプリストアから上記アプリをインストール
2. アプリの設定画面で商品をサブスクリプション対応に設定
3. 自動的に商品ページにサブスクリプションオプションが追加される

## ❓ よくある質問

### Q1: 商品ハンドルを変更できますか？
**A**: はい、商品編集画面の「検索エンジンリスト」セクションで変更できます。

### Q2: リンク先を外部URLにできますか？
**A**: はい、`href="/products/..."` を `href="https://外部URL"` に変更してください。

### Q3: ボタンの色やスタイルを変更したい
**A**: CSSの `.gc-btn-primary` や `.gc-btn-outline` クラスを編集してください。

### Q4: フリープランは¥0でも商品登録が必要？
**A**: 必要です。無料でも商品として登録することで、在庫管理や顧客管理ができます。
