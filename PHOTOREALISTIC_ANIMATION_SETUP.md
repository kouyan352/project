# 📸 実写風オープニングアニメーション セットアップガイド

## 🎯 概要

オープニングアニメーションが**CSS描画**から**本物の写真**を使用する実写版に変更されました。

### 変更前 vs 変更後

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| 視覚スタイル | CSSグラデーションで描いたイラスト | 実際の写真（フォトリアリスティック） |
| 手の表現 | CSS box-shadowで作ったシルエット | 本物の人間の手の写真 |
| 植物の表現 | border-radiusで作った簡易図形 | 実際の植物の写真 |
| 画像管理 | コードに埋め込み | Shopify管理画面から簡単アップロード |

---

## 📋 必要な写真（3枚 + オプション1枚）

### 1. 枯れた植物の写真
- **用途**: Scene 1 - 問題提起シーン
- **推奨サイズ**: 1000px × 1200px（縦長）
- **内容**: 
  - 葉が枯れている、元気のない植物
  - 暗めの背景
  - 鉢植えがはっきり見える
- **例**: しおれた葉、茶色くなった葉先、弱々しい姿

### 2. スマホを操作する手の写真
- **用途**: Scene 2 - 解決方法シーン
- **推奨サイズ**: 1200px × 1600px（縦長）
- **内容**:
  - **実際の人間の手**がスマホを持っている様子
  - 横から撮影（斜め角度）
  - スマホ画面が見える構図
  - 自然な持ち方
- **例**: 親指でスマホをタップしている手、LINEアプリを操作しているような構図

### 3. 元気な植物の写真
- **用途**: Scene 3 - 成功シーン
- **推奨サイズ**: 1000px × 1200px（縦長）
- **内容**:
  - 健康で生き生きとした植物
  - 明るい背景
  - 緑が鮮やか
  - 葉がピンと立っている
- **例**: 新芽が出ている、ツヤのある緑の葉

### 4. 机の背景写真（オプション）
- **用途**: Scene 3の背景
- **推奨サイズ**: 1920px × 400px（横長）
- **内容**:
  - 木製の机の天板
  - 自然光が当たっている
  - シンプルで清潔感
- **注意**: 未設定の場合は茶色の背景が表示されます

## 🚀 Shopifyでの設定手順

### ステップ1: 写真を準備

1. 上記の3枚（+1枚）の写真を用意
2. ファイル名を分かりやすく変更（例: `wilted-plant.jpg`, `hand-phone.jpg`, `healthy-plant.jpg`）
3. 推奨フォーマット: JPG または PNG

### ステップ2: Shopifyにアップロード

1. **Shopify管理画面**にログイン
2. **コンテンツ** → **ファイル** をクリック
3. **ファイルをアップロード** ボタンをクリック
4. 3枚の写真を選択してアップロード
5. アップロード完了を確認

### ステップ3: テーマにセクションを追加

1. **オンラインストア** → **テーマ** をクリック
2. **カスタマイズ** ボタンをクリック
3. トップページを開く
4. 左サイドバーで **セクションを追加** をクリック
5. **「Opening Animation（実写版）」** を選択
6. セクションをページの**一番上**に移動

### ステップ4: 写真を設定

セクション設定画面で以下を設定:

#### 📷 写真アップロード必須

1. **枯れた植物の写真**
   - 画像ピッカーをクリック
   - アップロードした枯れた植物の写真を選択

2. **スマホを持つ手の写真**
   - 画像ピッカーをクリック
   - アップロードした手とスマホの写真を選択

3. **元気な植物の写真**
   - 画像ピッカーをクリック
   - アップロードした元気な植物の写真を選択

4. **机の写真（オプション）**
   - 画像ピッカーをクリック
   - 机の写真を選択（スキップ可能）

#### ⚙️ アニメーション設定

- **オープニングアニメーションを有効化**: ✅ チェック
- **初回訪問時のみ表示**: お好みで設定
  - ✅ オン = 初回のみ表示（2回目以降は非表示）
  - ☐ オフ = 毎回表示

### ステップ5: プレビュー確認

1. 右上の **保存** をクリック
2. **プレビュー** ボタンで実際の表示を確認
3. スキップボタンの動作をテスト
4. スマホ表示も確認（レスポンシブ対応済み）

## 🎬 アニメーションの流れ

### 合計時間: 6.5秒

| 時間 | シーン | 内容 |
|------|--------|------|
| 0s - 1.5s | Scene 1 | 枯れた植物の写真 + 「元気がない...」テキスト |
| 1.5s - 3.5s | Scene 2 | 手とスマホの写真 + LINEチャット画面 |
| 3.5s - 6.5s | Scene 3 | 元気な植物の写真 + キラキラエフェクト |

### Scene 2の詳細（LINEチャット）

スマホ画面には以下のメッセージが表示されます:

1. **ユーザー**: 「葉が枯れてきました😢」
2. **タイピング中**: 「...」（3点アニメーション）
3. **専門家**: 「水やりを調整しましょう！💧 専門家がサポート」

## ⚠️ トラブルシューティング

### 問題: 写真が表示されない

**原因**: 画像が正しくアップロードされていない

**解決方法**:
1. Shopify管理画面 → コンテンツ → ファイル で画像を確認
2. セクション設定で画像ピッカーを再度開いて選択し直す
3. 保存してリロード

### 問題: 警告メッセージが表示される

**例**: 「⚠️ 枯れた植物の写真をアップロードしてください」

**原因**: その写真がまだ設定されていない

**解決方法**:
1. セクション設定を開く
2. 該当する画像ピッカーで写真を選択
3. 保存

### 問題: 画像のサイズが合わない

**症状**: 画像が切れている、または余白が多い

**解決方法**:
- 推奨サイズに合わせて写真を編集
- CSSの `background-size: cover` により自動調整されますが、推奨サイズに近いほど美しく表示されます

### 問題: アニメーションが表示されない

**確認事項**:
1. **トップページ（ホームページ）にいるか？**
   - アニメーションはトップページでのみ表示されます
2. **「初回訪問時のみ表示」がオンになっていないか？**
   - オンの場合、2回目以降は表示されません
   - リセット方法: ブラウザの開発者ツール → Application → Local Storage → `greencare_opening_animation_viewed` を削除

## 💡 写真撮影のコツ

### 枯れた植物
- 自然光で撮影（窓際）
- 背景をシンプルに（壁や机）
- 植物全体が入るように縦長で撮影
- 枯れている部分を強調

### スマホを操作する手
- 横から斜め45度の角度で撮影
- 手とスマホの両方がはっきり見えるように
- スマホ画面は明るく（実際の画面はCSSで上書きされます）
- 自然な手の形を意識

### 元気な植物
- 明るい場所で撮影（自然光が最適）
- 植物の緑を鮮やかに
- 葉のツヤや新芽が見えるように
- 清潔感のある背景

## 🔧 技術詳細

### 使用技術

- **Shopify Liquid**: テンプレート言語
- **CSS `background-image`**: 写真の表示
- **Shopify `image_picker`**: 管理画面での画像選択
- **CSS Animations**: シーン遷移とフェード効果
- **JavaScript**: タイミング制御とスキップ機能
- **localStorage**: 初回訪問の記録

### ファイル構造

```
shopify-theme/
└── sections/
    └── opening-animation.liquid  # 実写版アニメーション
```

### カスタマイズ可能な部分

コードを編集することで以下をカスタマイズできます:

- 各シーンの表示時間（デフォルト: 1.5s, 2s, 3s）
- 写真のサイズと配置
- テキストの内容（「元気がない...」など）
- LINEチャットのメッセージ内容
- スキップボタンのデザイン

## 📞 サポート

### よくある質問

**Q: CSS描画版に戻せますか？**
A: はい。Gitの履歴から以前のバージョンに戻すことができます。

**Q: 写真のファイルサイズに制限はありますか？**
A: Shopifyの制限に従います（通常は20MBまで）。ただし、ページ読み込み速度のため、各画像は1MB以下を推奨します。

**Q: GIF動画は使えますか？**
A: はい。GIF形式の画像も使用可能です。ただし、CSSアニメーションとの組み合わせに注意が必要です。

**Q: 動画（MP4など）は使えますか？**
A: いいえ。現在の実装は静止画像のみ対応しています。動画対応には別途カスタマイズが必要です。
