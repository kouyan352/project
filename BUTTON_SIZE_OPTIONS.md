# 🎨 ボタンサイズ変更オプション

## 現在のサイズ

```css
プライシングカードのボタン:
  padding: 1.25rem 3rem (上下20px, 左右48px)
  font-size: 1.25rem (20px)
  
アウトラインボタン (フリープラン、プレミアム):
  padding: 0.875rem 2rem (上下14px, 左右32px)
  font-size: 1rem (16px)
```

---

## オプション1: もっと大きく（特大）

```css
padding: 1.5rem 4rem;      /* 上下24px, 左右64px */
font-size: 1.5rem;         /* 24px */
```

**イメージ**:
```
┌────────────────────────────────────────┐
│                                        │
│        このプランを選ぶ  ← 特大        │
│                                        │
└────────────────────────────────────────┘
```

**用途**: 
- CTAを非常に目立たせたい
- コンバージョンを最大化
- スマホでタップしやすく

---

## オプション6: プライマリだけ大きく

```css
/* スタンダードプラン（人気）だけ大きく */
.gc-pricing-card.featured .gc-btn {
  padding: 1.5rem 4rem;
  font-size: 1.5rem;
}

/* フリーとプレミアムは標準サイズ */
```

**イメージ**:
```
┌─────────────┐  ┌──────────────────────────┐  ┌─────────────┐
│ フリー      │  │                          │  │ プレミアム  │
│             │  │  このプランを選ぶ ← 大   │  │             │
│ [小ボタン]  │  │                          │  │ [小ボタン]  │
└─────────────┘  └──────────────────────────┘  └─────────────┘
                      ↑ 人気プランを強調
```

**用途**:
- 推奨プランを目立たせる
- 視覚的な階層を作る

---

## オプション7: カスタムサイズ

お好みのサイズを指定：

```
padding: [上下]rem [左右]rem;
font-size: [サイズ]rem;

例:
padding: 1.2rem 3.2rem;
font-size: 1.3rem;
```

---

## 📱 モバイル対応

モバイルでは自動的に小さく調整されます：

```css
@media (max-width: 480px) {
  .gc-btn {
    padding: 1rem 2rem;
    font-size: 1rem;
  }
}
```
